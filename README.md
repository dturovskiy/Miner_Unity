# Miner_Scnenario

Суть гри: знайти цінний самородок який складається із чотирьох блоків і розміщений на карті під змлею.
Розмір карти: 250 х 100 плиток. Кожна плитка 128 пікселів.

В грі є дві сцени:
1. MenuScene
2. MainScene на якій відбувається весь ігровий процес

## MenuScene
Стартова сцена в якій є кнопки: 
- Вихід
- Налаштування
- Старт гри
